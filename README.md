# Diplom 1
Юнит тесты для классов Burger, Bun, Ingredient, IngredientType сервиса Stellar Burgers.

## Технологии

| Технология            | Версия |
|-----------------------|--------|
| Java                  | 11     |
| Mockito               | 5.16   |
| bite_buddy            | 1.17.2 |
| JUnit                 | 4.13.1 |
| Maven Compiler Plugin | 3.14.0 |
| JaCoCo Maven Plugin   | 0.8.12 |

## Запуск тестов

Для запуска тестов используйте команду:

```bash
  mvn verify
```
